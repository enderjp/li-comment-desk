# Comment Desk

Aplicacion web construida con Vite + React + TypeScript que consume Supabase como base de datos/autenticacion.

## Desarrollo local

1. Instala dependencias:
   `npm install`
2. Crea tu archivo `.env` usando `.env.example`.
3. Inicia el proyecto:
   `npm run dev`

## Variables de entorno

La app cliente usa estas variables:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
